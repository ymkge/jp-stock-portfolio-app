# 国内株式ポートフォリオ管理アプリ (jp-stock-portfolio-app)
<img width="1500" height="852" alt="スクリーンショット 2026-06-03 12 29 29" src="https://github.com/user-attachments/assets/ccb43e24-364c-46a7-9b96-19e0c13261a7" />
 
## 概要

ユーザーが管理する国内株式、米国株式、投資信託のポートフォリオに基づき、株価や財務指標、配当履歴などをYahoo!ファイナンスから取得し、Webページに一覧表示するシングルページアプリケーション（SPA）です。
各指標が割安か割高かを視覚的に判断するためのサポート機能も搭載しています。

## 主な機能

- **銘柄一覧表示**: 登録された銘柄の各種情報をリアルタイムで取得し、テーブル形式で表示します。
- **決算月表示**: 国内株式・米国株式について、ポートフォリオ一覧に各銘柄の決算月を表示し、権利確定日などの参考情報として活用できます。
- **複数口座対応の保有情報管理**: 1つの銘柄に対して、複数の口座（特定、一般、旧NISA, 新NISAなど）での保有情報を個別に管理できます。口座ごとに取得単価と数量を記録し、評価額、損益、税金等を自動計算し、年間配当予想を自動で計算・表示します。
- **ポートフォリオ分析**: 保有銘柄のみを対象とした分析ページで、ポートフォリオ全体のサマリー、保有口座ごとの詳細一覧、業種別・口座種別・**証券会社別**・国別ごとの保有比率を円グラフで視覚的に確認できます。さらに、**ポートフォリオ全体の「DNA（加重平均指標）」や「分散度（HHI指数）」、景気特性やスタイルを判定する「性格診断」機能**により、資産全体の傾向を多角的に分析可能です。
- **業種別KPIサマリー [新機能]**: [改善] 銘柄詳細一覧に加え、業種単位での運用効率を一望できる「業種別KPI」タブを追加しました。
    - **KPI比較**: セクターごとの総損益率、配当利回り、構成比などを横並びで比較・ソートでき、どのセクターが資産成長を牽引しているかを即座に特定可能です。
    - **ドリルダウン連動**: 業種名をクリックすることで、即座に該当業種の銘柄詳細一覧へジャンプ（フィルタリング）できるシームレスな分析動線を提供します。
- **プライバシー保護**: ページ上のすべての金額表示（評価額、損益など）をワンクリックで隠す「目隠し機能」を備えており、業種別KPIテーブルにも連動します。
- **指標ハイライト**: PERやPBRなどの指標が、設定ファイル(`highlight_rules.json`)の基準に応じて色付けされ、割安・割高が一目で分かります。
- **総合評価スコア**: 複数の指標（PER, PBR, ROE, 利回り, **連続増配**, **トレンド**, **フィボナッチ**, **RCI**）を総合的に評価した「スコア」を**最大15点**の星印で表示し、銘柄の魅力を多角的に判断できます。
    - **色分け表示**: 指標の種類に応じて星の色を分けて表示します。
        - **ファンダメンタルズ評価**（PER, PBR, ROE, 利回り, 連続増配）: **黄色**の星（★）
        - **トレンド評価**（移動平均線との乖離など）: **青色**の星（★）
    - **Wフィボ強調表示 [新機能]**: 短期と長期のフィボナッチ押し目が一致する「Wフィボ」発生時は、フィボナッチ列に専用バッジを表示し、視覚的にチャンスを強調します。
- **ポートフォリオ特性分析 (Portfolio DNA & Risk)**: [強化]
    - **ポートフォリオのDNA**: 保有額の大きさを考慮した「加重平均」により、資産全体の平均PER、PBR、ROE、利回りを算出。
    - **初心者向け最適化**: 専門用語を「割安さ」「稼ぐ力」などの直感的な日本語に翻訳し、基準値に基づき信号機カラー（青・黄・赤）で状態を表示。
    - **データの信頼性チェック**: 各指標の「カバー率」を算出し、データ不足（70%未満）時は「⚠️参考値」として警告を表示することで、誤った判断を防止。
    - **分散度・リスク分析**: [強化] 特定銘柄への集中度を測る「HHI指数（カゴの分け具合）」と「上位5銘柄占有率」を算出。DNA診断と同じグリッドUIを採用し、目標値との比較や、ⓘアイコンホバーによる「保有比率トップ5銘柄」の表示により、リスクの要因を透明化しました。
- **ポートフォリオ性格診断 (Style Analysis)**: [強化]
    - **スタイル判定**: 保有銘柄の属性から「景気特性（守りに強い/景気に敏感）」「投資スタイル（割安/成長）」「時価総額（大型/中小型）」の比率を可視化。
    - **ひとこと診断（アドバイス）**: 分析結果に基づき、「現在は大型の割安株が中心の、どっしりと安定した構成です」といった、現状を要約した日本語の診断メッセージを表示します。
- **ポートフォリオ特性 (Radar Chart)**: [強化]
    - **多角的視覚化**: 「割安性」「収益性」「インカム」「クオリティ」「モメンタム」「分散度」「安全性」の7つの軸で可視化。
    - **モメンタムの最適化**: [改善] モメンタム指標を順張り（トレンド追随）指標のみに最適化し、上昇相場での勢いを正しく評価します。
    - **DNA・トレンドの統合**: DNA分析によるPBR加重平均や、集中リスク（HHI指数）をチャートの計算に統合。資産全体の「強み」と「弱み」を多角的に把握できます。

    - **ベンチマーク比較**: 市場平均（ベンチマーク）を点線で重ねて表示し、自分のポートフォリオが市場に対してどのような特性を持っているかをひと目で把握できます。
- **連続増配分析**: 過去10年間の配当データから**連続増配年数**（前期比で増配している年数）を自動で計算し、バッジで表示します。詳細な配当履歴はツールチップで確認できます。
- **銘柄追加・削除**: 銘柄コードを指定して、ポートフォリオの銘柄を管理します。複数銘柄の一括削除、および直近追加した銘柄10件のリスト表示・入力補助機能も搭載しています。
- **ソート機能**: ポートフォリオ一覧および分析ページの保有銘柄一覧の各項目をクリックすることで、データを昇順・降順に並び替えることができます。銘柄コードや名称でのリアルタイムフィルタリング、**業種別フィルタ**、シグナルフィルタの組み合わせが可能です。
- **外部リンク**: 銘柄名や連続増配バッジをクリックすると、Yahoo!ファイナンスの該当ページを新しいタブで開きます。
- **フルワイドUI & インテリジェント・フィルタ (UI/UX Refresh)**: [強化]
    - **サイドバー廃止とフルワイド表示**: 画面を広く使うためサイドバーを廃止し、17列ある国内株テーブルの視認性を最大化しました。
    - **上部管理・ナビゲーションバーの最適化**: [改善] 銘柄追加フォームに加え、ボタン配置を直感的に再編。「分析ページ」への遷移を強調（塗りつぶしボタン）し、「全件更新」を適切な視認性の枠線ボタンに変更しました。
    - **ページ間UIの統一**: [改善] 分析ページのヘッダーUIをメインページと同期（🌓トグルスイッチの採用、戻るボタンのボタンスタイル化）。アプリ全体で一貫した操作感を提供します。
    - **分析ページ互換のフィルタ配置**: 検索窓、業種フィルタ、シグナルフィルタをテーブル直上に配置。分析ページと共通の操作感を提供します。
    - **統合シグナルフィルタ**: 買い場、格安、過熱などのシグナルを1つのドロップダウンに統合。
- **クイック詳細リンク (Quick Links)**: [新機能] メインページの国内株式一覧に、Yahoo!ファイナンスの「適時開示」と「業績詳細」へ直接ジャンプできるリンクバッジを追加。下落時の原因調査を劇的に高速化します。
- **CSVダウンロード**: 表示しているポートフォリオ全体や、分析ページの保有銘柄一覧をCSVファイルとしてダウンロードできます。
- **購入・状況注目フラグ (Market Signal)**: [強化] 時系列データに基づき、テクニカルとファンダメンタルズを高度に組み合わせた売買シグナルを自動判定します。
    - **マルチウィンドウ判定**: [新機能] フィボナッチ（押し目）判定を「1年」「半年」「3ヶ月」の3つの期間で同時に行い、短期的なチャンスも逃さず検知します。
    - **フィボナッチ・コンバージェンス (Wフィボ) [新機能]**: 短期（3/6ヶ月）と長期（1年）の押し目ラインが重なる「強いサポート帯」への到達を検知。この場合、シグナルを自動的に **Level 2 (🔥 チャンス)** へ昇格させます。
    - **リバウンド・トレンド転換検知 [強化]**: 25日移動平均線の突破や、**中期ゴールデンクロス (25/75)**、**長期ゴールデンクロス (75/200)** を捉え、反転の初動やトレンド形成を通知します。
    - **トレンド方向の明示 [強化]**: 購入シグナルに対し、株価が75日移動平均線より上（📈順張り）か下（⚡逆張り）かをラベルに表示。リスク度合いを直感的に把握可能にしました。
    - **リスク抑制ロジック**: 長期下降トレンド（200日線乖離）にある銘柄を「⚠️長期下降トレンド」として自動的に格下げし、「過熱・見送り」フィルタで一括除外可能です。
    - **判定精度の向上**: `sync_history` により蓄積された長期データと、**仮想挿入（Virtual Insertion）ロジック**を用いることで、通信なしでも正確なクロス判定とシグナル更新が可能になりました。
- **分析シグナル履歴の永続化**: [新機能] 銘柄ごとのスコアや売買シグナル（注目、チャンス、長期調整等）を、その判定時点のスナップショットとしてDBに自動保存。過去の特定の日に「どの銘柄にどのようなシグナルが出ていたか」を後から分析できるようになりました。
- **資産履歴の自動記録と詳細同期 (Sync History)**: [強化] 分析ページを開くたびに月次資産推移を保存する機能に加え、日次の株価・指標データを時系列で自動的にデータベースへ蓄積。同期ツール(`sync_history.py`)に `--force-resync` オプションを搭載し、株式分割等による過去データの不整合を容易に修復可能です。
- **月別配当分布 (予測)**: [新機能] 銘柄ごとの決算月と年間配当金に基づき、将来1年間の配当受け取りスケジュールを月別に可視化（棒グラフ）。キャッシュフローの把握を容易にします。
- **確定データ優先保存**: [新機能] 履歴データ保存時、ザラ場中の未確定な当日値を除外（前日までを厳守）することで、正確な終値ベースの時系列データを担保します。
- **比較対象日の可視化 [新規]**: 分析ページのサマリー比率について、比較対象となっている過去データの日付をヘッダー横やツールチップに明示するようにしました。
    - **フロントエンドキャッシュ**: APIから取得した資産データをブラウザにキャッシュし、再読み込み時の表示を高速化します。
- **スマート・キャッシュ・フェッチ & 自己修復分析**: [強化] バックエンド側でメモリキャッシュに加え、SQLiteによる日次永続化キャッシュを導入。市場の状態に合わせた最適なキャッシュ管理に加え、**ロジック変更時に過去データを自動的に再計算（自己修復）**して反映する機能を備えています。
- **コアロジックの自動テスト [強化]**: [Issue #1 完了] `pytest` を用いた自動テスト基盤を構築。スコアリング、売買シグナル（GC/DC含む）、資産計算、スクレイピングのパースロジック等、アプリの中核機能を網羅し、デグレを防止します。
- **データ更新レポート**: [新機能] データ取得完了後に、所要時間、成功/失敗数、アセットタイプ別の内訳、更新時刻を画面上に表示。システムの動作状況を直感的に把握できるようになりました。

- **データ整合性の保護 (File Locking)**: [新機能] `portfolio.json` への書き込み時に排他制御（ファイルロック）を導入。複数タブでの操作や同時リクエストによるデータ破損を防止し、資産情報の安全性を高めました。
- **エラーハンドリングの高度化とユーザー通知 [新機能]**: [Issue #3 完了] スクレイピング失敗時の原因（403制限、404コード不正、500エラー等）を詳細に判別し、UI上で具体的な対処法（待機、修正、再試行）を提示します。
    - **詳細エラー通知**: テーブルのエラー行において、詳細な理由と対処法をセル内およびツールチップで確認可能です。
    - **インテリジェント・クールダウン**: 403アクセス制限を検知した場合、全件更新ボタンを自動的に15分間ロックし、カウントダウンタイマーを表示することで、制限の悪化を物理的に防止します。

## 詳細な機能一覧

### 1. ポートフォリオ管理機能
- **資産の追加**
    - 国内株式、米国株式、投資信託を銘柄コード、ティッカー、ファンドコードでポートフォリオに追加する。
    - コードの形式から資産タイプ（国内株/米国株/投資信託）を自動で判別する。
- **資産の削除**
    - ポートフォリオから特定の資産を選択して一括で削除する。
- **資産情報表示**
    - 資産タイプごとにタブで表示を切り替える。
    - 各資産の最新情報を一覧で表示する（株価、前日比、時価総額など）。
    - 一覧の各項目でソート（並び替え）が可能。
    - 銘柄コードや名称で表示内容をフィルタリングする。
- **保有状況の管理**
    - 資産ごとに、複数の口座（特定口座, 一般口座, 新NISA, 旧NISA）の保有情報を登録・更新・削除できる。
    - 保有情報には「取得単価」と「数量」を記録する。
- **データ更新**
    - 「全件更新」ボタンで、ポートフォリオ内の全資産の情報を最新化する（クールダウン機能付き）。
- **CSVエクスポート**
    - 現在表示しているポートフォリオの一覧をCSVファイルとしてダウンロードする。

### 2. ポートフォリオ分析機能
- **サマリー表示**
    - ポートフォリオ全体の評価額、投資額、損益、損益率、年間配当合計などを表示する。
    - 金額表示のON/OFFを切り替えることができる。
- **グラフ表示**
    - 保有資産の構成比を円グラフで可視化する（業種別、口座種別、証券会社別、国別、配当構成）。
    - **ポートフォリオ特性（レーダーチャート）**: 「割安性」「収益性」「インカム」「クオリティ」「モメンタム」「分散度」「安全性」の7つの軸で資産全体のバランスを可視化。
    - **月別配当分布 (予測)**: 日本株・米国株の支払いパターンに基づき、将来の配当収入を月別に棒グラフで可視化。
- **保有銘柄一覧**
    - 全ての保有資産の詳細な情報を一覧で表示する（評価額、損益、損益率など）。
    - 銘柄名、業種、口座種別、証券会社、**シグナル**でフィルタリングが可能。
- **分析結果のCSVエクスポート**
    - 保有銘柄一覧をCSVファイルとしてダウンロードする。

### 3. データ取得・加工機能
- **スクレイピング（安定化・高速化対応済み）**
    - Yahoo!ファイナンスから以下の情報を取得する。
        - **国内株式**: 株価、企業情報（業種、決算月）、財務指標（PER, PBR, ROE, EPS）、配当情報（利回り, 過去配当履歴）。
        - **米国株式**: 株価、市場情報、財務指標（PER, EPS）、決算月。
        - **投資信託**: 基準価額、純資産総額、信託報酬。
    - **安定化対策**: `requests.Session` による接続維持、同時実行数の制限、銘柄内待機時間を導入し、アクセス制限（500エラー）を回避します。
    - **高速化対策**: キャッシュ存在時の待機スキップロジックにより、2回目以降の表示を高速化します。
- **スコアリング（国内株式のみ）**
    - PER, PBR, ROE, 配当利回り, 連続増配年数などの指標に基づき、独自の評価スコアを算出する。
- **注目シグナル判定**
    - テクニカル指標（RSI, RCI, フィボナッチ, 25日線等）とファンダメンタルズスコアを組み合わせ、買い場（**中期GC/25日線突破**等）や過熱感（**中期DC/25日線割れ**等）を判定する。

## セットアップと実行方法

1. **リポジトリをクローンします。**
   ```bash
   git clone https://github.com/your-username/jp-stock-portfolio-app.git
   cd jp-stock-portfolio-app
   ```

2. **必要なPythonライブラリをインストールします。**
   ```bash
   pip install fastapi uvicorn python-multipart requests jinja2 beautifulsoup4 cachetools pytest pytest-mock
   ```

3. **FastAPI開発サーバーを起動します。**
   ```bash
   uvicorn app:app --reload
   ```

4. **ブラウザでアクセスします。**
   Webブラウザを開き、 `http://127.0.0.1:8000` にアクセスしてください。
